# RestfulGloom

A web application for tracking your Gloomhaven campaign.

## Architecture

This is a REST API driven app, intending for easy integration with other Gloomhaven apps.

## Build and run

Pre-requisites:

- Java

then run with:

    ./mvnw spring-boot:run
